## [0.1.0] - 2026-01-04

### Added

- OpenTelemetry integration (`otel` package) with OTLP HTTP exporter
- Phoenix Cloud support via `PHOENIX_API_KEY` and `PHOENIX_SPACE_ID` environment variables
- OpenInference semantic conventions for LLM observability attributes
- Batch and sync span processing modes
- REST API client for Projects, Spans, Datasets, Experiments, Prompts, Annotations
- Evaluation framework (`evals` package) for running metrics on LLM outputs
- omniobserve/llmops Provider implementation for unified observability interface
- OpenInference span kinds: LLM, Chain, Tool, Agent, Retriever, Embedding, Reranker, Guardrail
- Prompt versioning and tagging support
- Span and trace-level annotations
- Integration tests for Phoenix Cloud
